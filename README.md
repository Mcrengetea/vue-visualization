# vite-visualization

#### Project composition

vite3 + vue3 + vue-router4 + element-plus + Threejs

#### Demo

https://mcrengetea.github.io/vue-visualization/

#### Project setup

```
npm install
```

#### Compiles and hot-reloads for development

```
npm run dev
```

#### Compiles and minifies for production

```
npm run build
```
